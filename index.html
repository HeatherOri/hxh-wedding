Our Story / 我們的故事

Location / 婚宴時間及地點

婚宴地點在景色怡人的士林青青食尚花園會館

想參與證婚儀式的朋友可以提前在11:30前入場
午宴實際開始時間為12:30

現場車位數量有限
搭乘大眾交通或接駁車會是比較理想的選擇
如果還是想要開車前往的朋友請記得預留時間哦！

婚宴日期 / 時間

  • 2023.12.30（六）

    • 11:00 - 12:00 迎賓茶會

    11:30 - 12:00 證婚儀式

    12:30 - 14:30 婚禮午宴

    婚宴地點

    青青食尚花園會館 凡爾賽廳

    臺北市士林區至善路二段266巷32號

    交通方式

  • 捷運/公車

    • 抵達捷運淡水線士林站，轉乘公車255、304、小型公車18、小型公車19、假日休閒公車101，至故宮博物院站下車

  • 青青接駁專車

    • 乘車處：士林捷運站一號出口直行至中正路左側（中正路249號）， 點我看乘車處地圖

    接駁時間：10:45 / 11:15 / 11:45 / 12:15

  • 自行開車（車位有限，較不建議）

    • 1. 南下：高速公路 → 堤頂大道下 → 右轉快速道路至內湖路一段 → 過自強隧道 → 故宮路與至善路口右轉 → 青青食尚花園會館

    2. 北上：高速公路 → 濱江交流道下 → 左轉大直橋 → 過自強隧道 → 故宮路與至善路口右轉 → 青青食尚花園會館

    3. 台北市東區 → 正氣橋 → 堤頂大道 → 過自橋隧道即可抵達

    4. 台北市北區 → 中山北路或文林路 → 中正路往外雙溪方向即可抵達

    5. 台北市區 → 復興北路機場地下道 → 過大直橋右轉 → 過自強隧道即可抵達

    Photos / 婚紗照先看

    別人總是說：
    「婚紗拍完就不會再拿出來看了！」

    所以我們想拍的是：自己會想一看再看的、每張照片都有自己故事的婚紗照

    因此，除了大堡礁潛水和夜間灑水攝影是請厲害的攝影朋友拍攝
    其他照片都是我們抱著超難行動的長紗裙、揹著重重的相機和腳架
    一邊旅行一邊紀錄的照片

    或許不如相館拍得華麗、妝髮也都是最輕描淡寫的那種

    但這就是最真實的我們！

    歡迎大家先以電子形式瀏覽我們精選的婚紗照
